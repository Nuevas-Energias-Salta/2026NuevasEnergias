## Visión General

Construir un **flywheel de crecimiento** que conecte la adquisición de clientes → venta → posventa → crosselling → expansión de productos, apalancado por un **Agente de IA** que automatice la gestión energética y genere nuevos leads de forma continua.

```
         │                                      │
         │         ┌───────────┐                │
         │         │  GROWTH   │                │
         │         │ (centro)  │                │
         │         └───────────┘                │
         │                                      │
  ┌──────┴───────┐     ┌──────────┐     ┌──────┴───────┐
  │ DESARROLLO   │←────│CROSSELL  │←────│  AGENTE IA   │
  │ PRODUCTOS    │     │B2C→B2B   │     │              │
  └──────────────┘     └──────────┘     └──────────────┘
```

## Pilar 1: 📣 Prospección

**Objetivo**: Generar un flujo constante de leads calificados a través de múltiples canales.

### Canales Activos

| Canal | Táctica | Frecuencia | KPI |
|-------|---------|------------|-----|
| **WhatsApp Campañas** | Mensajes segmentados a base de datos existente | Semanal | Tasa de respuesta > 15% |
| **LinkedIn** | Publicaciones de contenido + prospección directa | 3x/semana | Conexiones calificadas/semana |
| **Eventos** | Participación en ferias, Vistage, cámaras empresariales | Mensual | Leads por evento |
| **Campañas IG** | Ads + contenido orgánico en Instagram | Continuo | CPL (Costo por Lead) |
| **Workspace FB** | Comunidad y contenido en Facebook | Continuo | Engagement rate |

### Contenido (Nutrición de Leads)

| Formato | Descripción | Frecuencia | Objetivo |
|---------|-------------|------------|----------|
| **Newsletter / Blog** | Artículos sobre eficiencia energética, ahorro, tendencias | Quincenal | Authority building + SEO |
| **Vistage / Eventos** | Presentaciones para CEOs y decisores | Mensual | Leads B2B de alto valor |

### Proceso de Calificación
1. **Captura** → Lead entra por cualquier canal
2. **Calificación** → Score basado en: tamaño de empresa, consumo energético, urgencia
3. **Follow-up** → Contacto directo dentro de las 24hs
4. **Asignación** → Comercial asignado según tipo de lead (B2C/B2B)

### Métricas Clave
- 🎯 **Leads generados/mes** por canal
- 🎯 **Tasa de calificación** (leads calificados / leads totales)
- 🎯 **Tiempo de follow-up** (objetivo: < 24hs)
- 🎯 **Costo por lead** por canal

## Pilar 3: 🔁 Posventa y Gestión Energética

**Objetivo**: Fidelizar clientes, generar valor continuo y crear oportunidades de crosselling.

### Servicio de Gestión Energética

| Acción | Frecuencia | Canal | Automatizable |
|--------|------------|-------|---------------|
| **Envío de informes de consumo** | Mensual | Email / WhatsApp | ✅ Agente IA |
| **Alertas de anomalías** | Tiempo real | Push / WhatsApp | ✅ Agente IA |
| **Análisis de datos energéticos** | Trimestral | Reunión + informe | ✅ Parcial |
| **Identificación de oportunidades** | Continuo | CRM / Agente IA | ✅ Agente IA |
| **Generación de leads de consultoría** | Continuo | Automático | ✅ Agente IA |

### Proceso de Posventa
1. **Instalación completa** → Onboarding del cliente al sistema de monitoreo
2. **Primer informe** (mes 1) → Benchmarking de consumo
3. **Informes periódicos** → Datos de ahorro, producción solar, ROI
4. **Alertas proactivas** → El Agente IA detecta anomalías y alerta al cliente
5. **Revisión trimestral** → Reunión de seguimiento + identificación de nuevas oportunidades

### Métricas Clave
- 🎯 **NPS** (Net Promoter Score) > 70
- 🎯 **Tasa de retención** anual > 90%
- 🎯 **Upsell rate** (% de clientes que compran algo más en 12 meses)
- 🎯 **Leads de consultoría generados** desde posventa

## Pilar 4: 🔄 Crosselling — B2C a B2B

**Objetivo**: Convertir clientes residenciales en puertas de entrada a proyectos B2B de mayor valor.

### Estrategia

```
Cliente B2C (Base Instalada) 
    │
    ├─ Base PRE (500+ clientes) 🔥
    │   → Alto consumo eléctrico por calefacción
    │   → Solución 1 (Ticket Bajo): Upgrade Termostatos Smart (Eficiencia sin obra)
    │   → Solución 2 (Ticket Alto): FV On-Grid Balance Neto ("Calefaccioná Gratis")
    │
    ├─ WA campaña: "Danos la facturación de tu empresa"
    │   → Análisis energético gratuito
    │   → Propuesta de eficiencia para la empresa
    │
    └─ Crosselling de temporadas
        → Verano: Bombas de calor piletas + Solar
        → Invierno: Biomasa (Ñuke/Ofen) + Backup
```

### Tácticas
1. **Campaña WA segmentada**: Identificar clientes B2C que son empresarios/gerentes
2. **Propuesta**: "Ya ahorrás en tu casa, ¿por qué no en tu empresa?"
3. **Análisis gratuito**: Solicitar factura energética de la empresa
4. **Crosselling estacional**: Ofertas alineadas con la temporada

### Métricas Clave
- 🎯 **Conversión B2C→B2B** (% de clientes B2C que generan un lead B2B)
- 🎯 **Ticket B2B promedio** vs. B2C
- 🎯 **Tiempo de conversión** B2C → primer proyecto B2B

## Pilar 6: 🤖 Agente IA

**Objetivo**: Automatizar la posventa, gestión energética e identificación de oportunidades comerciales.

### Funcionalidades del Agente

| Función | Input | Output | Impacto |
|---------|-------|--------|---------|
| **Envío de informes** | Datos de consumo del cliente | Informe personalizado (PDF/WA) | Fidelización |
| **Alertas automáticas** | Anomalías en consumo | Notificación al cliente + comercial | Retención |
| **Análisis de datos** | Histórico de consumo | Insights de ahorro, benchmarking | Valor agregado |
| **Identificación de oportunidades** | Perfil del cliente + consumo | Score de oportunidad + producto sugerido | Revenue |
| **Generación de leads de consultoría** | Análisis de cartera | Leads priorizados para consultores | Growth |

### Flujo del Agente IA en el Flywheel

```
Datos del cliente → Agente IA → {
  → Informe mensual al cliente (retención)
  → Alerta si hay anomalía (cuidado proactivo)
  → Detección de oportunidad (crossell)
  → Lead para consultoría (nuevo negocio)
  → Input para prospección (referidos)
}
```

### Stack Técnico Sugerido
- **Monitoreo**: Integración con inversores/medidores solares
- **Análisis**: Scripts Python con modelos de consumo
- **Comunicación**: WhatsApp API + Email automatizado
- **CRM**: Integración para seguimiento comercial
- **Dashboard**: Web app para clientes (diagnostics app ya en desarrollo)
